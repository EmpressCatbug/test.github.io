Technology Stack

The backend is being developed using FastAPI for high-speed web handling, Uvicorn as the ASGI server, and Pydantic for robust data validation. The app is containerized using Docker for consistent deployment across platforms.

  • FastAPI: Python-based API framework (async-friendly)
  • OpenAI API: Summarization powered by GPT models
  • crawl4ai: Custom scraping library (in testing)
  • Docker: Used for development, deployment, and scaling
