Overview

This 3D graphics project began as a final assignment in my CS 330: Computational Graphics and Visualization course. I created a fully interactive OpenGL scene replicating my desk setup ‚Äî featuring a lavender corner desk, three monitors, and keyboard. The scene included WASD movement, perspective switching, textured surfaces, and multiple lighting types.

To improve visual quality and workflow, I enhanced the project by recreating the scene in Blender. This transition allowed me to incorporate high-resolution textures, advanced lighting, animation, and camera control.

Reflection

One of the most rewarding aspects of this project was developing the camera control system in OpenGL. By handling input buffers and transformation matrices directly, I created a responsive and immersive navigation system ‚Äî allowing users to move freely with the keyboard (WASD), look around with the mouse, and dynamically adjust speed and viewpoint with scroll and key toggles.

This low-level implementation gave me hands-on experience with mathematical foundations of 3D graphics, including view matrices, field of view adjustments, and event polling. It also helped reinforce best practices in modular programming, which later made the Blender transition easier.

Iteration, modularity, and user-centered interaction design were core to my process ‚Äî skills that will carry forward in future UI/UX or 3D development roles.

This project directly aligns with outcomes in software design, innovation, and visual communication.
